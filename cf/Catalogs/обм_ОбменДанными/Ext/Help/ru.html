Это сущность по перекидыванию между двумя базами.

Алгоритмы

Инициализация соединения

Функция которая соединяется со стороней базой.

Доступные переменные:

Обязательное заполнение:

Соединение - произвольный, в эту переменную необходимо вернуть значение соединение с внешней базой (например COM соединение).

 

Инициализация запроса HTTP

Функция которая соединяется со стороней базой.

Доступные переменные:

НаборДанных - ссылка на справочник набор данных, который иницилизирует запрос

стрТелоЗапроса - текст, тело запроса для HTTP запроса

Обязательное заполнение:

ЗапросHTTP- тип запрос HTTP который будет использоватся далее в коде
